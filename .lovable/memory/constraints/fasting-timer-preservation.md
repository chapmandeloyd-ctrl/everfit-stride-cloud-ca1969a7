---
name: Lion Fasting Timer Preservation
description: The lion fasting timer (FastingProtocolCard) is the signature feature of KSOM-360 and must NEVER be removed, replaced, or visually demoted. All new dashboard ideas must coexist with it.
type: constraint
---

The lion fasting timer (rendered by `FastingProtocolCard` on `ClientDashboard`) is the signature, non-negotiable centerpiece of the KSOM-360 client experience.

**Hard rules:**
- NEVER remove the lion timer from the dashboard
- NEVER replace it with an empty-state / "Choose Plan" hero illustration when no fast is active — the lion must always be visible
- NEVER demote it below the fold or hide it behind a tab
- NEVER swap the lion mark for generic illustrations (food, leaves, etc.)
- New plan-selection / onboarding flows (e.g. Simple-style "Start Here" cards, fasting plan browsers) must live ALONGSIDE the timer, not replace it — typically as a separate route like `/client/choose-protocol`

**Why:** The user has explicitly stated "I will never want to get rid of my lion fasting timer." It is core to brand identity and not subject to redesign.
